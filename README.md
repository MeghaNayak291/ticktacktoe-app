# Pink Tic Tac Toe

Pink Tic Tac Toe is a web-based game built using HTML, CSS, and JavaScript.  
It features a pink-themed interface and supports both single-player and two-player modes.

## Features

- Single-player mode (Play against Bot)
- Two-player mode (Local multiplayer)
- Win detection
- Draw detection
- Restart game option
- Responsive design

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Run

1. Clone the repository:
   git clone https://github.com/MeghaNayak291/ticktacktoe-app.git

2. Open the project folder.

3. Open index.html in your browser.
